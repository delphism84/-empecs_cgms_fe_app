1) 캡처 완료 화면

기능번호 목적 화면구성설명 관련 함수 스크린샷

ST_01_01 앱 설정(단위/시간형식/접근성 등) 사용자 카드, 일반 설정, 단위/알림/접근성 섹션 구성 SettingsPage.build(), SettingsPage._save()

3) 비고

  • 이번 결과는 에뮬레이터 실구동 후 하단 탭 기준 자동 캡처를 우선 반영했습니다.
  • 상세 페이지(로그인/NFC/QR/워밍업/이벤트 편집 등)는 라우트 자동 이동 안정화 후 2차 캡처로 완전판 제작 가능합니다.
  • XLSX/PPTX는 원하시면 다음 단계에서 Node 스크립트로 변환 생성 가능합니다.
